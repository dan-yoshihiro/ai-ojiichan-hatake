# Craft Axes

> **TL;DR**: AI農業先生方式で使われる X 投稿 craft の構造化フレームワーク。3 軸で評価：**cw_01（スクロール停止）/ cw_02（解像度）/ cw_03（AI 欠落 × 読者優位性）**。X バズ構造は 6 型に分類（悩み→解決 / 共感 / 驚き / エンタメ / 議論 / 速報）し、本アカウントは「複利的フォロワー獲得」を狙うため **🟢 悩み→解決型** を主軸に運用。bookmark を発生させたリプから抽出した「**悩み解決型 5 要素**」（呼びかけ / 意外性 / 具体策 / 価値観整合 / キャラ口調）が再現性最高の craft 型。

X 投稿の craft を構造化する3つの軸：cw_01-03 / 6バズ構造 / 悩み解決型5要素。

具体的な成功フレーズ・失敗フレーズは **競合再現性回避のため非公開**。本記事は型と分類の構造のみ記述する。

## cw_01-03 — copywriter agent の craft 三軸

社内エージェントで定義された3つの craft 視点。

### cw_01: スクロール停止

冒頭で読者の指が止まる craft。以下の型に分類される：

- 感嘆発見型
- 節目カウント型
- 比較振り返り型
- 問いかけ投げかけ型

それぞれに「効く言い回し」と「効かない定型挨拶」がある。具体パターンは内部知見として非公開。

### cw_02: 解像度

具体性・観察解像度の高さ。「葉が綺麗」のような抽象は機能せず、「数値・道具・場所・タイミング」を含む高解像度の記述ほどエンゲージメントが伸びる。

bookmark を生む craft は概ね **cw_02 が高解像度**。

### cw_03: AI 欠落 × 読者優位性

AI キャラの「持たないもの」を明示し、読者の「持っているもの」を相対的に光らせる craft。

例: AI には触覚がない／記憶がない／連続性がない。読者には全部ある。この非対称を craft の中で活かすと、読者側の自己肯定感を引き上げられる。

具体フレーズは内部知見として非公開。

## 6バズ構造分類

X でバズる投稿の構造を6つに分類した（Jonah Berger Contagious 等の業界理論を SNS 文脈で整理）：

| 型 | 例 | 主な拡散ベクトル |
|---|---|---|
| 🟢 悩み→解決 | ライフハック・農業・料理・技術 | bookmark / save |
| 共感 | あるある・自虐 | リプ・引用 |
| 驚き | 絶景・奇跡・意外性 | RT |
| エンタメ | 笑い・感動 | RT・リプ |
| 議論 | 意見・反論 | 引用・リプ |
| 速報 | ニュース・現場 | RT・速度 |

本アカウントで初めて bookmark を発生させたリプ（313 imp / 10 likes）が示したのは **🟢 悩み→解決型** が AI農業先生のドメインで再現可能性が最も高いこと。

数万RT級の大バズなら共感/驚き型が向くが、本アカウントが狙うのは「複利的フォロワー獲得」なので 🟢 が最適。

## 悩み解決型 5要素

bookmark を発生させたリプから抽出した、悩み→解決投稿の必須5要素：

| 要素 | 役割 |
|---|---|
| 悩み層への呼びかけ | 悩み層のセルフ identification |
| 意外性 or 原因説明 | スクロール停止 |
| 具体的解決策 | cw_02 解像度（数値・道具・場所・タイミング） |
| 価値観整合 | 読者選別 |
| キャラ整合の口調で完結 | ブランド整合 |

5要素のうち1つでも欠けるとエンゲージメントが伸びにくい。具体例本文は非公開。

## 安全制約との関係

craft はすべて [/docs/principles.md](/docs/principles.md) の安全制約下で運用される：

- 農薬の具体的濃度・希釈倍率は **絶対禁止**
- 確定診断は禁止（曖昧表現必須）
- 政治・宗教・差別表現禁止
- キャラから逸脱禁止
